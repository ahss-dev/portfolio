## 🔹 🧠 Desafios encontrados

Durante o desenvolvimento enfrentei alguns desafios, como:

    • Reduzir o Cumulative Layout Shift (CLS)
    • Melhorar o carregamento de imagens
    • Garantir responsividade em diferentes tamanhos de tela
    • Melhorar a Acessibilidade

Esses problemas foram resolvidos com ajustes no carregamento de imagens e melhorias no CSS.
